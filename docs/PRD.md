# 產品需求文件 (PRD)

**專案名稱**：機車保養與改裝紀錄系統
**專案描述**：專為熱血車主打造的機車管理系統，協助車主精準記錄名下車輛的日常保養里程、耗材更換狀況與花費，同時提供追蹤未來改裝願望清單的功能。

## 1. 專案概述

### 背景與動機
許多騎乘 DRG 2代、R15M 或黃紅牌重機的熱血車主，對愛車的保養與升級十分講究。然而，傳統記事本或普通的記帳軟體難以將「里程數」、「特定耗材（鍊條、機油、空濾等）」與「改裝品願望」有效結合。因此，需要一款專屬的機車管理系統，讓車主能有系統地追蹤車況、保養歷史與預估開銷。

### 目標用戶
需要精準管理機車日常保養、追蹤車輛花費，以及熱愛車輛升級（改裝）的熱血車主。

### 核心價值主張
- **精準追蹤**：綁定里程與保養項目，不再忘記何時該保養。
- **改裝預算管理**：集中管理心儀的改裝品與預估花費，讓升級更有計畫。
- **專屬車庫**：支援多車輛管理，從白牌到紅牌皆可輕鬆掌握。

## 2. 功能需求

### 1. 車輛管理 (個人車庫)
- **作為** 熱血車主，**我希望** 能夠新增、編輯與刪除我名下擁有的機車資訊（如廠牌、型號、目前里程數），**以便** 將不同的保養紀錄分別歸戶管理。

### 2. 保養紀錄追蹤
- **作為** 熱血車主，**我希望** 能夠針對特定車輛新增並追蹤保養項目（如換機油、空濾、鍊條清潔等）及花費，**以便** 了解愛車的保養歷史與耗材更換狀況。

### 3. 里程與保養到期提醒
- **作為** 熱血車主，**我希望** 能夠為每項耗材設定建議的保養週期（里程數），並在輸入或檢視當下里程時標示即將到期的項目，**以便** 提前安排保養，避免車輛受損。

### 4. 升級與改裝願望清單
- **作為** 熱血車主，**我希望** 能夠建立心儀改裝品的願望清單並輸入預估花費，**以便** 掌握總升級預算，有計畫地推進愛車的進化。

### 5. 總覽儀表板
- **作為** 熱血車主，**我希望** 在登入後能一眼看到「近期需更換保養項目」與「保養/改裝累積總花費」，**以便** 快速掌握目前的愛車狀況及財務支出。

## 3. 非功能需求

### 技術限制
- **後端框架**：Flask
- **模板引擎**：Jinja2
- **資料庫**：SQLite

### 效能與安全考量
- 系統應確保車輛數據與花費等個人紀錄的隱私，能正確區分不同使用者的資料（未來若有多人需求）。
- 所有表單輸入（如保養花費、里程數）需具備基礎防呆與驗證，避免輸入負數或不合理字元。
- 確保查詢資料庫紀錄時的速度，維持流暢的使用者體驗（頁面載入應即時）。

## 4. MVP 範圍

**【Must Have】必要功能**
- 新增、編輯、刪除個人車庫車輛。
- 記錄耗材保養（日期、項目、里程、花費）。
- 提供每個保養項目是否到期的提示（針對里程計算）。

**【Should Have】重要功能**
- 改裝品願望清單（項目、預估花費、是否已改裝之狀態標記）。
- 首頁總覽儀表板功能，用來彙整保養提醒。

**【Nice to Have】加分功能**
- 首頁圖表化呈現花費分析。
- 支持手機端的響應式網頁設計 (RWD)，方便車庫施工時紀錄。
